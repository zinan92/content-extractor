<objective>
Add hallucination detection heuristics and model updates to flag unreliable transcriptions.

Purpose: Whisper generates confident-sounding fabricated text on silence, music, and background noise (especially with Chinese audio from Douyin/XHS). Without detection, hallucinated transcripts poison downstream LLM analysis. This plan adds per-segment suspicion flags and transcript-level quality warnings.

Output: New hallucination.py module with detection heuristics, updated Pydantic models with warning fields, VideoExtractor wired to populate hallucination warnings.
</objective>

    - detect_repeated_ngrams(segments, n=4, threshold=3) -> list[str]: finds 4-grams appearing 3+ times across all segment text, returns the repeated phrases
    - check_transcript_hallucinations(segments, speech_ratio) -> tuple[str, ...]: returns tuple of warning strings combining all heuristics:

      - "Low overall confidence ({avg:.2f}) -- review recommended" if avg confidence < 0.5
      - "Repetitive text detected -- possible hallucination: {phrases}" if repeated n-grams found
      - "{N} of {M} segments flagged as suspicious" if any segments are suspicious
