    1. Create `src/content_extractor/loader.py` per 01-RESEARCH.md Pattern 5:
       - Define `ContentItemNotFoundError(Exception)` with message containing the missing path
       - Define `ContentItemInvalidError(Exception)` with message containing JSON parse error details
       - `load_content_item(content_dir: Path) -> ContentItem`:
         - Build `item_path = content_dir / "content_item.json"`
         - If not exists: raise ContentItemNotFoundError(f"No content_item.json in {content_dir}")
         - Read bytes with `item_path.read_bytes()`
         - Parse with `orjson.loads(raw)` in try/except orjson.JSONDecodeError -> raise ContentItemInvalidError
         - Validate with `ContentItem.model_validate(data)` in try/except ValidationError -> raise ContentItemInvalidError
         - Return the validated ContentItem
