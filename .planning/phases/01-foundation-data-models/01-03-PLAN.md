  <action>
    1. Create `src/content_extractor/extract.py`:
       ```python
       from dataclasses import dataclass
       from pathlib import Path
       import time
       import sys

       from content_extractor.config import ExtractorConfig
       from content_extractor.loader import load_content_item, ContentItemNotFoundError, ContentItemInvalidError
       from content_extractor.router import get_extractor, UnsupportedContentTypeError
       from content_extractor.output import write_extraction_output, is_extracted

       @dataclass(frozen=True)
       class BatchError:
           content_dir: str
           error: str

       @dataclass(frozen=True)
       class BatchResult:
           succeeded: tuple  # tuple of ExtractionResult
           failed: tuple     # tuple of BatchError
           total: int
           success_count: int
           failure_count: int
       ```

       - `extract_content(content_dir: Path, config: ExtractorConfig | None = None) -> ExtractionResult | None`:
         - config defaults to ExtractorConfig() if None
         - If is_extracted(content_dir) and not config.force_reprocess: return None (skipped)
         - load_content_item(content_dir) -> item
         - get_extractor(item.content_type) -> adapter
         - adapter.extract(content_dir, config) -> result
         - write_extraction_output(content_dir, result, item, force=config.force_reprocess)
         - return result

       - `extract_batch(parent_dir: Path, config: ExtractorConfig | None = None) -> BatchResult`:
         - config defaults to ExtractorConfig() if None
         - Find all dirs containing content_item.json: `sorted(p.parent for p in parent_dir.rglob("content_item.json"))`
         - For each dir, try extract_content. On success, append to succeeded list. On ANY exception, append BatchError to failed list, print error to stderr, continue (per D-07, QUAL-01).
         - Return BatchResult with totals.
